# EL-IoT
Demonstrating the concept of ensemble learning 

Models are in `.\models`

Datasets:
`> https://www.kaggle.com/datasets/syedsaqlainhussain/sql-injection-dataset`

`> https://www.kaggle.com/datasets/sajid576/sql-injection-dataset`
